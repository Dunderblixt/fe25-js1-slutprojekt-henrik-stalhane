A simple project using TMBDs API to make searches for Movies, TV-shows or people in showbuisness.
TMDB API Documentation: https://developer.themoviedb.org/reference/collection-details.
the library Animejs is used to bring the webpage's main Headline to life using animation.
Library documentation: https://animejs.com/documentation/.
